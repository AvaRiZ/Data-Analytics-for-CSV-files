# Data-Analytics-for-CSV-files
A web-based application for data analytics and visualization built with Streamlit. This tool allows users to upload CSV files, perform exploratory data analysis, and create interactive visualizations without writing any code.

# Features
# Data Upload & Preview

<h5>-Upload CSV files via drag-and-drop or file browser</h5>
<h5>-Preview uploaded data with pagination support </h5>
<h5>-Automatic detection of data types and formats </h5>

# Data Analysis

<h5>-Basic Statistics: View summary statistics (mean, median, mode, standard deviation)</h5>
<h5>-Data Overview: Check data dimensions, column information, and missing values </h5>
<h5>-Correlation Analysis: Generate correlation matrices for numerical columns </h5>
<h5>-Data Cleaning: Handle missing values and filter data based on conditions </h5>

# Visualization Tools

<h5>-Scatter Plots: Visualize relationships between two numerical variables</h5>
<h5>-Line Charts: Plot trends over time or sequential data </h5>
<h5>-Bar Charts: Compare categorical data across groups </h5>
<h5>-Histograms: Analyze distribution of numerical data </h5>
<h5>-Box Plots: Identify outliers and understand data spread</h5>
<h5>-Heatmaps: Visualize correlation matrices </h5>
<h5>-Pie Charts: Show proportional data (for categorical variables) </h5>

# Interactive Features
<h5>-Customizable chart parameters (colors, titles, labels)</h5>
<h5>-Real-time filtering and sorting</h5>
<h5>-Export visualizations as PNG images</h5>
<h5>-Download processed data as CSV </h5>

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
# How to Use
<h5>1. Upload your CSV file using the uploader in the sidebar</h5>
<h5>2. Explore your data in the "Data Preview" section</h5>
<h5>3. Analyze statistics in the "Data Analysis" section</h5>
<h5>4. Create visualizations by selecting chart types and customizing parameters</h5>
<h5>5. Export results or download processed data</h5>
